# 时间线功能 - 快速开始

## 🚀 5分钟快速体验

### 1. 启动后端服务
```bash
cd backend
python start_dev.py
```

### 2. 打开微信开发者工具
- 打开项目：`wechat-miniprogram`
- 确保"不校验合法域名"已勾选

### 3. 登录小程序
- 如果未登录，先完成登录流程

### 4. 进入时间线
- 点击底部tabBar的"时间线"图标（⏰）

### 5. 查看时间线
- 浏览你的所有记录
- 尝试筛选功能
- 点击记录查看详情

---

## 📱 主要功能

### 1. 查看记录
- 按日期分组显示
- 左侧时间轴可视化
- 每条记录显示完整信息

### 2. 筛选记录
- **类型筛选**：全部、进度、里程碑、困难
- **时间范围**：最近7天、30天、90天、全部

### 3. 查看详情
- 点击任意记录卡片
- 跳转到详情页

### 4. 刷新数据
- 下拉刷新最新记录
- 上拉加载更多历史

## 🔍 如何测试

### 准备测试数据
1. 创建几条语音记录
2. 创建几条拍照记录
3. 创建几条文字记录
4. 标记一些为"重要"或"里程碑"

### 测试筛选
1. 点击"进度"标签
2. 点击"里程碑"标签
3. 切换时间范围

### 测试交互
1. 下拉刷新
2. 上拉加载更多
3. 点击记录查看详情

## 🐛 遇到问题？

### 数据为空
- 确保已创建过程记录
- 检查筛选条件
- 尝试选择"全部"类型和"全部"时间范围

### 加载失败
- 确保后端服务正在运行
- 检查网络连接
- 查看控制台错误信息

### 样式异常
- 清除缓存重新编译
- 检查WXSS文件是否正确加载

## 💡 使用技巧

1. **快速筛选**：使用类型标签快速查找特定类型的记录

3. **重点关注**：重要记录会有特殊标记和颜色
4. **详情查看**：点击记录可查看完整信息
